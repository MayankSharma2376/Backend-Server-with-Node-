# Backend Server with Node.js

A simple backend server built using **Node.js** that demonstrates the fundamentals of backend development — routing, serving static files, and handling requests. This project is part of my journey into backend engineering, moving from REST APIs and MVC architecture into database integration.

---

## 📂 Project Structure
Backend-Server-with-Node- │
├── css/             # Stylesheets
├── data/            # Data files
├── img/              # Images 
├── views/            # HTMLviews
├── logEvent.js       # Logging utility
├── server.js         # Main server file
├── package.json      # Project metadata & dependencies
└── .gitignore        # Ignored file




---

## 🚀 Features
- Serves **HTML pages** dynamically based on routes  
- Handles **static assets** (CSS, images)  
- Includes a **logging utility** (`logEvent.js`) for tracking server activity  
- Lightweight and beginner-friendly — ideal for learning backend basics
